# WWE Fantasy Live UI Upgrade

This update improves the no-login live Supabase app layout.

## New UI additions
- Player display-name banner
- Cleaner home page
- Better league hub
- Separate solo and multiplayer draft sections
- League cards show join code, draft status, and season
- Cleaner display-name/profile page
- Improved News & Rumors page with quick-read panels
- Live source check remains available

## Supabase
